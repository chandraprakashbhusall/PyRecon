# PyRecon 🔍

A Python-based network reconnaissance tool built to understand how port scanning and service detection work internally.

# Features

- Multi-threaded port scanning
- Open / Closed / Filtered port detection
- Banner grabbing
- JSON report generation
- HTML report generation
- MITRE ATT&CK mapping (T1595 - Active Scanning)
- CLI-based usage


# Example to run code:
python main.py 192.168.1.254 -p 1-1000 -d 0.05
